The following function takes any number under one quadrillion and finds the path to 4 using the "cosmic algarithm". The "cosmic algarithm" means each number is mapped to the number of letters in it's english name. Example: the cosmic path of 18 is 18>8>5>4 and 4 is cosmic, meaning it maps to itself (and is the only number to do so in english.)
